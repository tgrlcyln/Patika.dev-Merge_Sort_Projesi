# Merge Sort Projesi

Proje 2

[16,21,11,8,12,22] -> Merge Sort

1.Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
2.Big-O gösterimini yazınız.

## Aşamalar :

[16,21,11,8,12,22]   
[16,21,11] - [8,12,22]] 
[16,21] - [11] -- [8,12] - [22]
[16] [21] - [11] --- [8] [12] - [22] 
[11,16,21] --- [8,12,22]
[8,11,12,16,21,22]
## Big O Notation Gösterimi :

Worst Case: O(nlogn)
Average Case: O(nlogn)
Best Case: O(nlogn)
